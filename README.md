# Post-Parent Support System (親亡き後支援エージェント)

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Status: Production](https://img.shields.io/badge/Status-Production-green)

**「親がいなくなったら、誰がこの子のことを一番に考えてくれるの？」**

このシステムは、そんな切実な問いに応えるために開発された、オープンソースの自律型AIエージェントチームです。
支援団体やNPOが、専門的なIT知識なしで、障害のある当事者のための「デジタル後見人」を構築・運用できるように設計されています。

---

## 🌟 特徴：3人のAIエージェントチーム

> [!WARNING]
> **免責事項**: 本システムは支援者の意思決定をサポートするためのものであり、**医療行為や診断を行うものではありません**。
> 医学的な判断が必要な場合は、必ず有資格者（医師・看護師等）に相談してください。

あなたのPCの中で、3人の専門エージェントが24時間体制で連携して働きます。

1.  **📝 Input Agent (書記・受付)**
    *   日々の支援日誌や報告を読み込み、整理整頓してデータベースに記録します。
    *   「この書き込みはプライバシー的に大丈夫？」などの安全チェックも行います。

2.  **🐕 Emergency Watchdog (番犬・SOS担当)**
    *   すべての報告をリアルタイムで監視します。
    *   「パニック」「発作」などの緊急事態を検知すると、即座に医師や家族への連絡プラン（SOS）を提示します。

3.  **🧠 Clinical Advisor & Support Agent (臨床参謀)**
    *   **Clinical Advisor**: 行動障害（拒否・自傷など）に対し、世界中の最新研究(PBS)と、本人の過去の記録を照らし合わせ、最適な対応策をアドバイスします。
    *   **Support Agent**: 「お母さんならこうしていた」という過去の膨大な記録から、「今の場面に最適なケア」を提案します。

## 🛠 導入方法

難しいインストール作業は不要です。**Docker** という技術を使って、コマンド一つで全ての環境が整います。

### 必要環境
- Docker Desktop (Windows / Mac)

### 起動手順

1. このプロジェクトをダウンロードします。
2. ターミナル（またはPowerShell）で以下のコマンドを実行します。

```bash
docker-compose up --build
```

これだけで、システムが起動します。
ブラウザで `http://localhost:8501` にアクセスすれば、そこはもうエージェントたちの待機所です。

## 🛡 プライバシーと安全性

- **ローカル完結**: データはすべてあなたのPC内（Dockerコンテナ内）のNeo4jデータベースに保存されます。クラウドに個人情報が漏れることはありません。
- **匿名化機能**: AIが処理する前に、自動的に個人名や電話番号をマスキングする機能を備えています。
