# Post-Parent Support System v1.0.0-beta Release Notes

## 🚀 はじめに
「親亡き後支援エージェント」のベータ版（v1.0.0-beta）をリリースしました！
このシステムは、ご家族や支援者が入力する日常の記録をAIが分析し、将来や緊急時に役立つ「構造化された知識」として蓄積するデータベースです。

専門知識は不要です。**「ダウンロードして、ダブルクリックするだけ」** で使い始められます。

## ✨ 主な機能
- **3人のAIエージェント**: 書記、見守り番犬、臨床アドバイザーが24時間体制でサポート。
- **ハイブリッド運用**: 日常は見守りモード（自律型）、分析したい時はClaude Desktop連携（分析モード）が可能。
- **ワンクリック起動**: 難しいコマンド入力は不要。

## 📥 インストール方法

1.  このページの **Assets** から `Source code (zip)` をダウンロードして解凍してください。
2.  フォルダの中にある起動スクリプトをダブルクリックしてください。
    - **Windowsの方**: `start.bat`
    - **Macの方**: `start.command`
    - ※ 初回は Docker Desktop のインストール等が必要な場合があります。画面の指示に従ってください。
3.  設定ウィザードが起動するので、質問に答えて設定を完了させてください。

## ⚠️ 注意事項
- **医療免責**: 本システムはアドバイスを提供しますが、医療診断ではありません。
- **ベータ版**: 予期せぬ不具合が含まれる可能性があります。重要なデータは必ずバックアップを取ってください。
