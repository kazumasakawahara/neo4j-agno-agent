# 「親なき後」の不安と向き合う ― Neo4j Agno Agent が目指すもの

## はじめに ― 現場で聞いた、ある母親の言葉

「この子のことを一番知っているのは私たち親です。でも、私たちがいなくなったら、誰がこの子のことを分かってくれるんでしょうか」

計画相談支援の仕事を続ける中で、何度この言葉を聞いてきたことでしょう。知的障害のあるお子さんを持つ親御さんの多くが、この「親なき後」という言葉にならない不安を抱えています。それは単なる経済的な心配ではありません。何十年もかけて積み上げてきた「我が子のトリセツ」が、自分たちがいなくなった瞬間に消えてしまうのではないか。その恐怖なのです。

## 知的障害者支援の現状と課題

知的障害のある方への支援は、極めて個別性の高いものです。同じ診断名であっても、一人ひとりの特性は全く異なります。パニックになったときの対処法、食事の好み、コミュニケーションの取り方、落ち着ける環境。それらは長い時間をかけて、主に親御さんが試行錯誤の中で見つけ出してきた「暗黙知」です。

しかし現状では、この貴重な知識は親御さんの頭の中にしか存在しません。紙の記録があったとしても、緊急時にすぐに参照できる形にはなっていないことがほとんどです。

私が最も危機感を覚えるのは、緊急時の対応です。ある利用者さんがパニックを起こしたとき、その場にいた支援者が良かれと思って後ろから声をかけました。しかし、その方にとってそれは最もやってはいけない対応でした。結果、パニックは悪化し、本人も支援者も傷つくことになりました。もし「後ろから声をかけてはいけない」という情報が即座に共有されていれば、この二次被害は防げたはずです。

支援者が変わるたびに、ゼロから関係性を構築し直さなければならない。手帳や受給者証の更新期限を誰も管理できていない。親御さんが倒れたとき、誰に連絡すればいいか分からない。これらは全て、情報が適切に記録・共有されていないことに起因する課題です。

## なぜこのプロジェクトを立ち上げたのか

私がこのプロジェクトを始めた理由は単純です。テクノロジーの力で、親御さんの「暗黙知」を形にできるのではないか。そう考えたからです。

近年のAI技術の発展は目覚ましいものがあります。自然言語で質問すれば、必要な情報を即座に引き出せる。構造化されていない情報を、意味のある形に整理できる。これらの技術を福祉の現場に応用すれば、これまで不可能だったことが可能になるのではないか。

Neo4j Agno Agentの核心は、「本人の尊厳に配慮した支援」を実現することにあります。私たちは利用者さんを「管理対象」として扱うのではなく、歴史と意思を持つ一人の人間として記録します。好きなこと、嫌いなこと、大切にしている思い出、将来の願い。それらを丁寧に聞き取り、グラフデータベースという形で関係性を含めて保存します。

特に力を入れたのが「Safety First」というAI運用プロトコルです。緊急時には、まず「やってはいけないこと」を最優先で表示する。これにより二次被害を防ぎ、その場を落ち着かせるための具体的な対処法へと誘導します。支援者が「何をすべきか分からない」という状態を、テクノロジーの力で解消したいのです。

## このプロジェクトに期待すること

私がこのプロジェクトに期待することは、大きく三つあります。

一つ目は、親御さんの安心です。「自分たちがいなくなっても、この子の情報は残る」という安心感は、何物にも代えがたいものです。親御さんが元気なうちに、その知識を丁寧に記録していく。それ自体が、親御さんへの敬意であり、本人への最大の支援だと考えています。

二つ目は、支援の質の向上です。新しい支援者が入ったとき、AIに「この方について教えて」と聞けば、必要な情報が統合された形で提供される。引き継ぎの時間は大幅に短縮され、最初から質の高い支援が可能になります。

三つ目は、本人の権利擁護です。手帳や受給者証の更新期限を自動で管理し、期限切れによる不利益を防ぐ。これは地味な機能ですが、本人の権利を守るという意味では極めて重要です。

## さらなる発展性

このプロジェクトは、知的障害者支援にとどまるものではありません。

将来的には、地域の社会資源をグラフデータベースで可視化し、本人を中心としたエコマップを自動生成する機能を実装したいと考えています。また、複数の事業所が連携して一人の方を支援する場合の情報共有基盤としても活用できるでしょう。

さらに、このシステムはオープンソースとして公開することを予定しています。福祉の現場でテクノロジーを活用したいと考える人々が、自由にカスタマイズして使えるようにしたい。そうすることで、日本中の「親なき後」の不安を、少しでも和らげることができればと願っています。

## おわりに

テクノロジーは万能ではありません。AIが人間の支援者に取って代わることもありません。しかし、テクノロジーは人間の支援をより良いものにする力を持っています。

「この子のことを知っている人がいる」という安心。それを、親御さんだけでなく、本人にも、支援者にも、地域にも広げていくこと。それが、Neo4j Agno Agentが目指す未来です。

一人の計画相談支援専門員として、この挑戦を続けていきたいと思います。
