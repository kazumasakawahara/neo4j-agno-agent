# 「自分のことは自分で決める」を支える ― Neo4j Agno Agent が目指すもの

## はじめに ― 当たり前の権利を、当たり前に

「自分のことは自分で決める」

これは、全ての人に等しく保障されるべき権利です。知的障害があろうとなかろうと、この権利に違いはありません。しかし現実には、知的障害のある方の意思決定は、しばしば周囲の人々によって代わりに行われてしまいます。「本人のため」という善意のもとで。

私がこのプロジェクトを立ち上げた最大の理由は、知的障害のある方が自分自身の人生を自分で選び取っていける環境を作りたい、ということに尽きます。そしてそのためには、支援に関わる全ての人が「この方には自己決定の権利がある」ということを深く自覚する必要があるのです。

## 自己決定を阻む壁 ― 情報の断絶という問題

知的障害のある方の自己決定を支えるためには、その方のことを深く理解している必要があります。どんな人生を歩んできたのか。何が好きで、何が苦手なのか。どんな場面で力を発揮し、どんな状況で困難を感じるのか。そして何より、本人が何を願っているのか。

これらの情報は、長い年月をかけて蓄積されてきたものです。幼少期から寄り添ってきた親御さんや兄弟。学校や施設で関わってきた先生や支援員。それぞれが、本人との関わりの中で得た貴重な経験値を持っています。

しかし現状では、これらの情報は断片的にしか共有されていません。親御さんの頭の中にある「我が子の歴史」、支援者が積み上げてきた「うまくいった支援法」、本人が折に触れて口にしてきた「将来の夢」。これらがバラバラのままでは、本人の自己決定を適切に支えることはできません。

新しい支援者が関わるたびに、ゼロから関係を築き直す。本人の願いが十分に伝わらないまま、支援者の判断で物事が決められていく。これでは、本人が主体的に人生を選び取ることなど、到底できません。

## このプロジェクトが目指すもの

Neo4j Agno Agentは、本人を中心とした情報共有の基盤です。

生育歴をはじめとした本人の歴史。親や兄弟が長年の関わりの中で培ってきた経験値。支援の過程で見出されてきた効果的な関わり方。そして何より、本人自身の願い（wish）。これらを一つのデータベースに統合し、支援に関わる全ての人が共有できるようにします。

ここで強調したいのは、この情報共有の目的は「支援者が管理しやすくするため」ではないということです。目的はただ一つ。本人が自分の願いを、支援者の助けを借りながら、自分自身の力で実現できるようにすること。それだけです。

本人のwishがデータベースの中心にあります。「一人暮らしがしたい」「好きな仕事に就きたい」「友達と旅行に行きたい」。そうした願いに向かって、どんな支援が必要なのか。本人の特性や歴史を踏まえて、どんなアプローチが効果的なのか。支援者はAIの力を借りながら、本人と一緒に考えることができます。

## 社会の一員として生きるために

知的障害のある方は、「支援を受ける存在」である前に、「社会の一員」です。

地域で暮らし、働き、人と関わり、時に誰かの役に立つ。そうした当たり前の社会参加を実現するためには、本人を取り巻く環境全体が変わる必要があります。本人のことを理解し、適切な配慮ができる人が増えること。本人が自分の意思を表明しやすい環境が整うこと。そして、本人の決定が尊重される文化が根付くこと。

このプロジェクトは、その第一歩です。本人の情報が適切に共有されることで、支援者は「この方は何ができないか」ではなく「この方は何を望んでいるか」という視点で関わることができるようになります。支援の質が変われば、本人の可能性も広がります。

将来的には、地域の社会資源とのつながりも可視化していきたいと考えています。本人を中心としたネットワークを広げ、社会の中で自分らしく生きていくための土台を作る。それが、このプロジェクトの目指す姿です。

## おわりに ― 支援者への願い

支援に関わる全ての方にお願いしたいことがあります。

目の前にいる方には、自分の人生を自分で決める権利があります。たとえ言葉でうまく表現できなくても、たとえ判断に時間がかかっても、その権利は変わりません。私たち支援者の役割は、代わりに決めることではなく、本人が決められるように支えることです。

そのために必要な情報を、このシステムは提供します。本人の歴史、家族の経験、効果的な支援法、そして本人の願い。これらを手がかりに、本人と一緒に、本人の人生を作っていってください。

「自分のことは自分で決める」

この当たり前の権利が、当たり前に守られる社会へ。Neo4j Agno Agentは、その実現に向けた私たちの挑戦です。
